## ✨ Features

* 🔄 Auto-detects accepted submissions
* 💻 Supports LeetCode & HackerRank
* 📂 Pushes code directly to GitHub
* 📁 Organizes solutions by difficulty
* 🌐 Multi-language support
* ⚡ Instant sync after submission

## ⚙️ Installation

### 🔹 Manual (Developer Mode)

1. Download ZIP
2. Extract folder
3. Open browser:

   ```
   chrome://extensions/
   ```
4. Enable **Developer Mode**
5. Click **Load Unpacked**
6. Select project folder

---

## 🧪 Usage

1. Open extension popup
2. Enter:

   * GitHub Token
   * Username
   * Repo Name
3. Solve problem
4. Click **Submit**
5. 🎉 Code auto-pushed to GitHub

---

## 🔐 Security & Privacy

* 🔒 Token stored locally (`chrome.storage`)
* 🚫 No external data collection
* 🔗 Only communicates with GitHub API

---

## 📌 Requirements

* GitHub account
* Personal Access Token (`repo` permission)
